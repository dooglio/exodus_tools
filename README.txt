# exodus_tools

This respository is my personal collection of various simple, portable scripts and tools for crypto-wallets, key manipulations, etc.

BIP39: This tool is from https://github.com/iancoleman/bip39 full credit to iancoleman for making this fantastic tool!

verifyLinux.sh: A linux shell script to automatically verify a downloaded Exodus wallet. The script assumes your download is in your ~/Downloads folder. All you need is curl, gnupg, and the URL to the newest release hashes, available from www.exodus.io/releases/
